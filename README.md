# Python Practice Repository

A curated collection of Python practice exercises, organized by chapter for easy learning and review.

---

## 📁 Repository Overview

This repository contains example scripts and practice problems for beginner-to-intermediate Python concepts. Each chapter focuses on a core topic, with standalone files and small projects to help reinforce learning.

## 📚 Chapter Breakdown

- `chapter 1/` — Introduction and basic Python structure
  - `first.py`, `module1.py`
  - `practice set/` contains early practice problems
- `chapter 2/` — Variables, data types, and operators
  - `input.py`, `operators.py`, `type.py`, `variables-datatypes.py`
- `chapter 3/` — Strings and escape sequences
  - `string.py`, `string-functions.py`, `escape_sequence.py`, practice files
- `chapter 4/` — Lists and tuples
  - `list.py`, `list-methods.py`, `list-mutability.py`, `tuple.py`, `tuple_methods.py`
- `chapter 5/` — Dictionaries and sets
  - `dictionary.py`, `dictionary-methods.py`, `dict.py`, `set.py`, `operations-in-sets.py`
- `chapter 6/` — Conditional statements and flow control
  - `conditional-statement.py`, `nested_if.py`, practice files
- `chapter 7/` — Loops and iteration
  - `loops.py`, `for_loops.py`, `while_loop.py`, `break_continue_in_loop.py`, `pass.py`
- `chapter 8/` — Functions and recursion
  - `function.py`, `function_with_argument.py`, `default_parameter.py`, `return_in_function.py`, recursion scripts
- `chapter 9/` — File handling and input/output
  - `file.py`, `file_write.py`, `more_file_functions.py`, plus sample text files and a `tables/` folder
- `project 1/` — Project-style practice problems
  - `snake_water_gun.py` and other practical exercises

## 🚀 Getting Started

1. Open the repository in VS Code or another Python editor.
2. Choose a chapter folder to explore the included examples.
3. Run any Python file using:

```bash
python3 path/to/file.py
```

4. Modify the example files to test new ideas and reinforce concepts.

## 💡 Best Practices

- Begin with `chapter 1/` and move through the chapters sequentially.
- Use `practice*.py` files to check your understanding.
- Make small edits, then run the script again to see how changes behave.
- Keep notes on what each script is teaching you.

## 📝 Notes

- Each chapter is designed to build on previous lessons.
- `chapter 9/` includes text files used by file I/O examples.
- `project 1/` contains a small project for applying multiple concepts together.
