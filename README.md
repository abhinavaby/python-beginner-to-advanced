# Python Practice Repository

This repository contains a structured set of Python practice exercises organized by chapter. Each chapter folder includes example scripts and practice problems that follow an introductory Python curriculum, from basic syntax to file handling.

## Repository Structure

- `chapter 1/` - Introduction and basic program structure
  - `first.py`, `module1.py`
  - `practice set/` contains early practice problems
- `chapter 2/` - Variables, data types, and operators
  - `input.py`, `operators.py`, `type.py`, `variables-datatypes.py`
- `chapter 3/` - Strings and escape sequences
  - `string.py`, `string-functions.py`, `escape_sequence.py`
- `chapter 4/` - Lists and tuples
  - `list.py`, `list-methods.py`, `list-mutability.py`, `tuple.py`, `tuple_methods.py`
- `chapter 5/` - Dictionaries and sets
  - `dictionary.py`, `dictionary-methods.py`, `dict.py`, `set.py`, `operations-in-sets.py`
- `chapter 6/` - Conditional statements
  - `conditional-statement.py`, `nested_if.py`
- `chapter 7/` - Loops and iteration
  - `loops.py`, `for_loops.py`, `while_loop.py`, `break_continue_in_loop.py`, `pass.py`
- `chapter 8/` - Functions and recursion
  - `function.py`, `function_with_argument.py`, `default_parameter.py`, `return_in_function.py`, recursion scripts
- `chapter 9/` - File handling and I/O
  - `file.py`, `file_write.py`, `more_file_functions.py`, plus sample text files and a `tables/` directory
- `project 1/` - Project-style practice problems
  - Several practice scripts related to conditionals and branching logic

## How to Use

1. Open this repository in a Python-capable editor like VS Code.
2. Choose a chapter folder to explore the related examples and practice scripts.
3. Run Python files with:

```bash
python3 path/to/file.py
```

4. Modify or extend the practice scripts as needed to deepen your learning.

## Notes

- Most files are standalone scripts demonstrating Python concepts.
- `chapter 9/` contains text files used for file read/write examples.
- `project 1/` contains practice problems intended to combine earlier concepts.

## Recommended Workflow

- Start with `chapter 1/` and progress through the chapters in order.
- Use the `practice*.py` files to test your understanding.
- Experiment by editing scripts and running them again.
